:::prof
L'imperfetto décrit le passé habituel ou une description : 'Da bambino giocavo...' (enfant, je jouais...). Terminaisons : -avo/-evo/-ivo. Il exprime ce qu'on FAISAIT régulièrement, ≠ passato prossimo (action ponctuelle).
:::
