:::prof
Le passé coréen se forme avec '~았/었어요'. 가다 → 갔어요 (je suis allé), 먹다 → 먹었어요 (j'ai mangé). Règle : voyelle ㅏ/ㅗ → 았어요, sinon → 었어요. C'est une base fondamentale pour raconter son voyage.
:::
